FAQ

Common Queries Answered

Here are some answers regarding dynamic configurations, security protocols, and deployments.

Is my system data secure and private?
Yes. Leafy Dash operates on secure JWT authentication models, handles encrypted database connections, and enforces strict client/server validations to block cross-site scripting (XSS) and unauthorized requests.
How do reviews prevent rating spam or bombs?
The backend verifies if the client leaving a review has a logged transaction history linked to their customer email/phone record. If no invoice match is found, the submission is securely declined.
Can I toggle modules on and off dynamically?
Yes. In the Settings configuration module, you can turn modules (such as Inventory, Sales Log, QR Reviews) on or off. Our engine immediately updates the database flags and alters your sidebar UI on-the-fly.
Where is my store data stored when deployed?
By default, the project runs on local SQLite. When deployed to a server environment (like Render or Railway), we support PostgreSQL connections. Simply add your database URL to the server environment variables.
